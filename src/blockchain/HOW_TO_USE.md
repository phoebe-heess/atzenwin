# How to Use These Files

## What's Included
1. `interfaces/` - Contains all the TypeScript definitions
2. `docs/` - Contains detailed integration documentation
3. `services/` - Contains the service implementation placeholder

## Getting Started
1. Open `docs/INTEGRATION.md` first - this contains all the technical requirements
2. Review the interfaces in `interfaces/` folder
3. Use `services/plumeService.ts` as your implementation template

## Important Notes
- All files are written in TypeScript
- The interfaces define the exact structure we need
- The service file shows what methods need to be implemented
- Security requirements are detailed in the integration guide
